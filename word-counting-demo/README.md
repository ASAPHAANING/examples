# word-counting-demo

A simple word-counting demo to showcase [Neon](https://github.com/rustbridge/neon).

# Setup

See the [requirements for Neon](https://github.com/rustbridge/neon#requirements) (TL;DR: Node, Rust, and on OS X, XCode).

# Build

```bash
$ npm install
$ node -e 'require("./")'
```
